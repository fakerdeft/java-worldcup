## 기능 목록

1. 입력 뷰

- [ ] 출력하고자 하는 항목에 대해 입력.
    - [ ] 1,2,3,4,5이외의 값은 예외처리 후 다시 입력.
- [ ] 대소문자 구분 없이 8개의 조(A ~ H) 중 하나를 입력.
    - [ ] A~H 이외의 값은 예외처리 후 다시 입력.
- [ ] 월드컵 출전 국가의 이름을 입력.
    - [ ] 출전 국가 이외의 값은 예외처리 후 다시 입력.

2. 출력 뷰

- [ ] 게임 시작 문구 출력.
- [ ] 게임 종료 문구 출력.
- [ ] 예외 상황 시 "[ERROR]"로 시작하는 에러 문구 출력.
